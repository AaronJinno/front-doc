## Ⅱ. Vue打包部署

#### 1. 打包
使用build指令对项目进行打包。
```cmd
npm run build
```
打包后的目录默认就是dist目录。

#### 2. 同步到远程服务器
:::tip
暂时采用手动上传，或者git同步吧，其他方式比较复杂，以后会和Nginx配置一起更新。
:::
将打包后的目录dist，同步到远程的服务器指定目录（与Nginx的配置有关）

**同步的方式**  
- 手动上传：比较麻烦，且慢
- 使用git：以在git-bash或者powershell使用scp指令
    - 果是linux环境开发，还可以使用rsync指令
- 添加脚本：为了方便，可以在package.json脚本中加一个push命令
